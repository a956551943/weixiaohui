# 多人共享编辑设置（约 5 分钟）

三人打开**同一个链接**，勾选和填写内容会**实时同步**。

---

## 第一步：创建 Supabase 项目（免费）

1. 打开 https://supabase.com 注册登录  
2. **New Project** → 取名 `osaka-trip` → 设数据库密码 → 选离中国近的 region（如 Singapore）  
3. 进入项目 → **Settings → API**  
4. 复制：
   - **Project URL** → 填入 `config.js` 的 `supabaseUrl`
   - **anon public** key → 填入 `supabaseAnonKey`

---

## 第二步：建表

项目里打开 **SQL Editor**，粘贴执行：

```sql
create table if not exists trip_states (
  room_id text primary key,
  data jsonb not null default '{}',
  updated_at timestamptz default now()
);

alter table trip_states enable row level security;

create policy "anon_all" on trip_states
  for all to anon using (true) with check (true);
```

再打开 **Database → Publications**，确认 `supabase_realtime` 包含 `trip_states` 表。  
（若没有：SQL 里执行 `alter publication supabase_realtime add table trip_states;`）

---

## 第三步：开启同步

编辑 `docs/config.js`：

```javascript
window.TRIP_SYNC = {
  enabled: true,
  supabaseUrl: 'https://你的项目.supabase.co',
  supabaseAnonKey: '你的anon密钥',
  defaultRoom: 'osaka2026-wxh',
};
```

提交并 push 到 GitHub。

---

## 第四步：分享给同行人

把下面链接发到微信群（三人点同一个）：

## 说明

| 项目 | 说明 |
|------|------|
| 房间号 | URL 里 `?room=xxx` 相同的人共享一份数据 |
| 实时性 | 一人勾选，其他人约 1 秒内更新 |
| 隐私 | 知道链接的人都能编辑；房间号请用不易猜的名字 |
| 未配置时 | 仍可用，但仅保存在各自手机（单机模式） |
